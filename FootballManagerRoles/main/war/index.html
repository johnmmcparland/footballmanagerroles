Football Manager Roles

How To

Welcome to Football Manager Roles!

To use this software, use the "Print Screen" option from the "FM" menu in Football Manager.
You should do this when on a players "attributes" screen

  1. Start Football Manager
  2. Load an existing game or create a new one
  3. Click on any players name to get to the player screen
  5. Click on the "Attributes" button (see figure 1 below)
  6. Go to the FM menu
  7. Choose "Print Screen" (see figure 2 below)
  8. Choose Text File
  9. Save the file to a known location, use the default file name
  10. Use this file in the form below
